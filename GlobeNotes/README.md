# GlobeNotes

GlobeNotes ist eine Webanwendung zum Verwalten und Visualisieren von Reisezielen.

## Features
- Reiseziele hinzufügen, anzeigen, löschen
- Reiseziele inklusive Bild, Highlights, Jahr, Ort und Kategorie
- Anzeige der Reiseziele auf einer Weltkarte
- Upload und Speicherung von Bildern
- Dockerized Backend inklusive persistenter Speicherung

## Technologien
- Vite & React (Frontend)
- Spring Boot (Backend)
- Docker
- Leaflet (Kartenanzeige)
- Maven

## Projekt starten
### Backend
```bash
cd backend
mvn spring-boot:run
````

# Frontend

```bash
cd frontend
npm install
npm run dev
```

### Docker Compose

```bash
docker compose up
```
